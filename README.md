# Dotfiles

This git repo is heavily inspired by this [repo](https://github.com/Sin-cy/dotfiles).

# Install

Clone this repo.

**Important**
Make sure you have git installed(comes pre installed with macos).

```zsh
git clone https://github.com/vladgrskkh/dotfiles.git $HOME/dotfiles
```

Than go to created directory and make **install.sh** executable.

```zsh
cd dotfiles && chmod +x ~/dotfiles/install.sh
```

After that go ahead and run this script.

```zsh
/bin/bash ~/dotfiles/install.sh
```
